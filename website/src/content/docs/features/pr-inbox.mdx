# PR Inbox

The PR Inbox is the main interface of GitDesk, showing all your pull request activity in one place.

## Inbox Sections

### Awaiting Your Review

PRs where you've been requested as a reviewer:
- Shows PR title, repository, and author
- Displays time since review was requested
- Indicates CI/CD status

### Your Pull Requests

PRs you've created:
- Track review status from your team
- See merge readiness at a glance
- Monitor CI/CD results

### Assigned PRs

PRs assigned to you:
- View your assignments across repositories
- Track progress on assigned work

## PR Card Information

Each PR card displays:

| Field | Description |
|-------|-------------|
| **Title** | The PR title |
| **Repository** | Owner/repo format |
| **Author** | PR author with avatar |
| **Status** | Open, merged, or closed |
| **Reviews** | Approval status |
| **CI Status** | Build/test results |
| **Updated** | Time since last update |

## Actions

- **Click** - Expand PR details
- **Double-click** - Open in browser
- **Right-click** - Context menu options

## Refresh Behavior

- Auto-refresh every 5 minutes
- Manual refresh with the refresh button
- Pull-to-refresh on touch devices
