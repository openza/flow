Openza Flow is built with Flutter and can be compiled for Linux, macOS, and Windows.

## Prerequisites

- Flutter SDK 3.10.3 or later
- Dart SDK 3.0.0 or later
- Git

### Platform Dependencies

<Tabs>
  <TabItem label="Windows">
    - Visual Studio 2022 with "Desktop development with C++" workload
    - Windows 10 SDK
  </TabItem>
  <TabItem label="Ubuntu/Debian">
    ```bash
    sudo apt install clang cmake ninja-build pkg-config \
      libgtk-3-dev libsecret-1-dev libjsoncpp-dev libnotify-dev
    ```
  </TabItem>
  <TabItem label="Fedora">
    ```bash
    sudo dnf install clang cmake ninja-build \
      gtk3-devel libsecret-devel jsoncpp-devel libnotify-devel
    ```
  </TabItem>
  <TabItem label="Arch">
    ```bash
    sudo pacman -S clang cmake ninja gtk3 libsecret jsoncpp libnotify
    ```
  </TabItem>
</Tabs>

## Build Steps

<Steps>
1. **Clone the repository**
   ```bash
   git clone https://github.com/openza/flow.git
   cd flow
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Build the application**

   <Tabs>
     <TabItem label="Windows">
       ```bash
       flutter build windows --release
       ```
       The built application will be in `build/windows/x64/runner/Release/`.
     </TabItem>
     <TabItem label="Linux">
       ```bash
       flutter build linux --release
       ```
       The built application will be in `build/linux/x64/release/bundle/`.
     </TabItem>
   </Tabs>
</Steps>

## Creating Installers

### Windows Installer

```bash
dart run inno_bundle --release
```

### Linux AppImage

```bash
./scripts/build-appimage.sh
```

### Linux Flatpak

```bash
./scripts/build-flatpak.sh
```

## Development Mode

For development with hot reload:

<Tabs>
  <TabItem label="Windows">
    ```bash
    flutter run -d windows
    ```
  </TabItem>
  <TabItem label="Linux">
    ```bash
    flutter run -d linux
    ```
