# Welcome to Openza Flow

Openza Flow is a desktop application that brings your GitHub pull request review inbox to your desktop. Available for Windows and Linux. Stay on top of code reviews without constantly checking GitHub.

## Features

<CardGrid stagger>
  <Card title="Review Requests" icon="approve-check-circle">
    See all PRs where your review has been requested, organized and ready to review.
  </Card>
  <Card title="Your PRs" icon="document">
    Track pull requests you've created and their review status.
  </Card>
  <Card title="Desktop Notifications" icon="bell">
    Get notified when you're added as a reviewer or when your PRs get updates.
  </Card>
  <Card title="Quick Actions" icon="rocket">
    Open PRs directly in your browser with a single click.
  </Card>
</CardGrid>

## Quick Start
