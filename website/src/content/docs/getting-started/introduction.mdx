# Welcome to Openza GitDesk

Openza GitDesk is a desktop application that brings your GitHub pull request review inbox to your Linux desktop. Stay on top of code reviews without constantly checking GitHub.

## Features

<CardGrid stagger>
  <Card title="Review Requests" icon="approve-check-circle">
    See all PRs where your review has been requested, organized and ready to review.
  </Card>
  <Card title="Your PRs" icon="document">
    Track pull requests you've created and their review status.
  </Card>
  <Card title="Desktop Notifications" icon="bell">
    Get notified when you're added as a reviewer or when your PRs get updates.
  </Card>
  <Card title="Quick Actions" icon="rocket">
    Open PRs directly in your browser with a single click.
  </Card>
</CardGrid>

## Quick Start

1. [Install GitDesk](/gitdesk/getting-started/installation) on your Linux system
2. [Set up your GitHub token](/gitdesk/getting-started/github-token)
3. Start reviewing PRs with a beautiful desktop interface
