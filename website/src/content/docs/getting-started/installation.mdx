# Installation

Openza Flow is available for Windows and Linux.

## Download

<Tabs>
  <TabItem label="Windows">
    Download the Windows installer from [GitHub Releases](https://github.com/openza/flow/releases/latest).

    <Steps>
    1. Download the `Openza-Flow-*-Setup.exe` file
    2. Run the installer
    3. Follow the setup wizard
    4. Launch Openza Flow from the Start menu or desktop shortcut
    </Steps>

    **System Requirements:**
    - Windows 10 or newer (64-bit)
  </TabItem>

  <TabItem label="AppImage">
    Download the latest AppImage from the [GitHub Releases](https://github.com/openza/flow/releases/latest) page.

    <Steps>
    1. Download the `.AppImage` file
    2. Make it executable:
       ```bash
       chmod +x Openza-Flow-*.AppImage
       ```
    3. Run the application:
       ```bash
       ./Openza-Flow-*.AppImage
       ```
    </Steps>

    **System Requirements:**
    - Ubuntu 22.04+, Fedora 35+, or Debian 12+
    - GLIBC 2.34 or newer
  </TabItem>

  <TabItem label="Flatpak">
    Download the Flatpak bundle from [GitHub Releases](https://github.com/openza/flow/releases/latest).

    <Steps>
    1. Install Flatpak if not already installed:
       ```bash
       # Ubuntu/Debian
       sudo apt install flatpak

       # Fedora
       sudo dnf install flatpak
       ```
    2. Install Openza Flow:
       ```bash
       flatpak install Openza-Flow-*.flatpak
       ```
    3. Run the application:
       ```bash
       flatpak run com.openza.flow
       ```
    </Steps>
  </TabItem>
</Tabs>
